# Projet IaC AWS - S3 + Lambda + DynamoDB

## Objectif
Déployer une architecture simple sécurisée avec AWS CloudFormation : 
- Bucket S3 chiffré (SSE-S3)
- Lambda pour upload de fichiers
- DynamoDB pour stocker les métadonnées
- Pipeline CI/CD avec GitHub et CloudFormation

## Instructions
1. Cloner le repo
2. Exécuter `scripts/deploy.sh`
3. Vérifier les ressources sur AWS Console
